---
title: 'LlamaCpp'
---

You can also use locally running models by `node-llama-cpp`.

## Install LlamaCpp addon

```bash
npm install @cherrystudio/embedjs-llama-cpp
```

## Usage

```ts
import { RAGApplicationBuilder } from '@cherrystudio/embedjs';
import { LlamaCpp } from '@cherrystudio/embedjs-llama-cpp';

const app = await new RAGApplicationBuilder()
.setModel(new LlamaCpp({
    modelPath: "../models/llama-3.1-8b-instruct-hf-q4_k_m.gguf",
}))
```
