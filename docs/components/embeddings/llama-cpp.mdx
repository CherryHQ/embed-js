---
title: 'LlamaCpp'
---

EmbedJs supports local embeddings via `LlamaCpp`.

## Install LlamaCpp addon

```bash
npm install @cherrystudio/embedjs-llama-cpp
```

## Usage

```ts
import { RAGApplicationBuilder } from '@cherrystudio/embedjs';
import { LlamaCppEmbeddings } from '@cherrystudio/embedjs-llama-cpp';

const app = await new RAGApplicationBuilder()
.setEmbeddingModel(new LlamaCppEmbeddings({
    modelPath: "./models/nomic-embed-text-v1.5.f16.gguf",
}))
```
